# Diwali_Sales_Analysis
This project performs a comprehensive Exploratory Data Analysis (EDA) on Diwali sales data to uncover consumer purchasing patterns and improve business strategies

Project Overview
Using a dataset of over 11,000 entries, the analysis examines various factors influencing sales:
Demographics: Analyzes purchasing behavior across Gender and Age Groups (e.g., 0-17, 26-35).
Geography: Identifies high-performing States and Zones.
Socio-Economics: Evaluates the impact of Occupation and Marital Status on order amounts.
Product Performance: Tracks the popularity and revenue generated by different Product Categories, such as Auto and Healthcare.

Technical Implementation
The project is built in Python using standard data science libraries:
Pandas & NumPy: For data cleaning, handling duplicates, and managing the 15-column dataset.
Matplotlib & Seaborn: For creating visual insights, including count plots and bar labels to represent sales distributions effectively.
The final goal is to provide actionable insights that help a retail business optimize its inventory and marketing efforts during the festive season
